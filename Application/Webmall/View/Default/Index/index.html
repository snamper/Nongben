"侬本佳园"项目

/  物联网+互联网+农业

目前的传统农业，有着产品供求信息不对称、销售渠道不畅通等诸多因素制约着农村经济的发展。

本项目致力于构建基于物联网技术的农业电商平台，并将其推广到实际应用中。平台包括商城、农情资讯、农产品溯源、智慧农业四大模块，将传统农业注入物联网与互联网的基因。带给广大农民生产便利的同时，保障农产品消费者的食品安全。

生态农业

时空结构型生态农业

这是一种根据生物种群的生物学、生态学特征和生物之间的互利共生关系而合理组建的农业生态系统，使处于不同生态位置的生物种群在系统中各得其所，相得益彰，更加充分的利用太阳能、水分和矿物质营养元素，是在时间上多序列、空间上多层次的三维结构，其经济效益和生态效益均佳。具体有果林地立体间套模式、农田立体间套模式、水域立体养殖模式，农户庭院立体种养模式等。

食物链型生态农业

这是一种按照能量流动和物质循环规律设计的一种良性循环的农业生态系统。系统中一个生产环节的产出是另一个生产环节的投入，使得系统中的废弃物多次循环利用，从而提高能量的转换率和资源利用率，获得较大的经济效益，并有效的防止农业废弃物对农业生态环境的污染。具体有种植业内部物质循环利用模式、养殖业内部物质循环利用模式、种养加工三结合的物质循环利用模式等。

综合型生态农业

这是时空结构型和食物链型的有机结合，使系统中的物质得以高效生产和多次利用，是一种适度投入、高产出、少废物、无污染、高效益的模式类型。

"互联网+邮政农村电商"

在当前“大众创业，万众创新”的形势下，“互联网+邮政农村电商”成了广大村民就业谋业的手段。

项目要点

  • 云平台:建立信息服务平台，推送种子、种苗、农资和农产品的最新行情信息，发布新品种与配套栽培技术，帮助农户与农技员、专家建立一对一的服务，建立涉农人员社交平台，安全农产品溯源查询。
  • 物联网:通过与示范基地栽培参数对比，提高农产品品质与产量预期。同时点对点进行线上病虫害诊断等，为农业增效、农户增收做保障。
  • 互联网:农资网上商城，厂家一级资源入驻，厂家直供用户，提供价廉可靠农资商品及相关服务。也是平台主要盈利点。
  • 大数据:农业分析决策，数据库共享。
